# 🚂 Railway Environment Variable Update

## Новая переменная для разделения AI моделей

Мы разделили использование AI моделей для оптимизации производительности и стоимости:

- **Генерация экзаменов**: `gemini-2.5-flash-002` (качество)
- **Чат с тьютором**: `gemini-2.5-flash-lite` (скорость + экономия в 6 раз)

## 📝 Что нужно добавить в Railway

### Новая переменная окружения:

```
GEMINI_CHAT_MODEL=gemini-2.5-flash-lite
```

### Обновить существующую переменную:

```
GEMINI_MODEL=gemini-2.5-flash-002
```

## 🔧 Инструкция по добавлению в Railway:

1. Откройте ваш проект в Railway Dashboard
2. Перейдите в раздел **Variables**
3. Добавьте новую переменную:
   - **Name**: `GEMINI_CHAT_MODEL`
   - **Value**: `gemini-2.5-flash-lite`
4. Обновите существующую переменную `GEMINI_MODEL`:
   - **Value**: `gemini-2.5-flash-002`
5. Сохраните изменения
6. Railway автоматически перезапустит сервис

## ✅ Преимущества этого разделения:

- ⚡ **Быстрее**: Чат отвечает со скоростью 887 токенов/сек
- 💰 **Дешевле**: Экономия в 6 раз на чате ($0.40 vs $2.50 за 1M output токенов)
- 🎯 **Качественнее**: Экзамены генерируются более мощной моделью
- 📊 **Точнее**: Flash Lite показывает 93.35% точности на Superfacts бенчмарке

## 🔍 Проверка после деплоя:

После добавления переменных и перезапуска проверьте логи:
- Для генерации экзаменов должна использоваться `gemini-2.5-flash-002`
- Для чата должна использоваться `gemini-2.5-flash-lite`

Логи будут содержать строки типа:
```
[GeminiProvider] Calling gemini-2.5-flash-002 API...  # Для экзаменов
[GeminiProvider] Calling gemini-2.5-flash-lite API... # Для чата
```
