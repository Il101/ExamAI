# 🎓 Добро пожаловать в ExamAI!

Привет! Спасибо, что согласились помочь протестировать мой новый проект! 

## 🤔 Что это такое?

**ExamAI** — это умный помощник для подготовки к экзаменам, который использует искусственный интеллект (Google Gemini).

### Простыми словами:
Вы загружаете свои учебные материалы (конспекты, PDF-файлы), а приложение:
1. 📚 **Анализирует** материал и разбивает его на темы
2. 🤖 **Создаёт** карточки для запоминания
3. 🧠 **Помогает учить** с помощью умной системы повторений
4. 📊 **Отслеживает** ваш прогресс

## ✨ Главные возможности

- **AI-генерация контента** — ИИ создаёт учебные материалы из ваших файлов
- **Умные повторения** — система сама определяет, когда вам нужно повторить материал
- **Карточки** — интерактивные флешкарточки для запоминания
- **Аналитика** — графики прогресса и статистика обучения
- **Pomodoro таймер** — встроенный таймер для продуктивной учёбы
