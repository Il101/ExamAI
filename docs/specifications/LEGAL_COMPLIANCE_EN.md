## Data Retention Policy

### Aufbewahrungsrichtlinie

| Datentyp | Aufbewahrungsfrist | Rechtsgrundlage | Löschverfahren |
|----------|-------------------|-----------------|----------------|
| **Nutzerkonto (aktiv)** | Bis zur Kündigung | Vertragserfüllung | Soft Delete → Hard Delete (90d) |
| **Lernmaterialien** | Bis zur Löschung durch Nutzer | Vertragserfüllung | Sofortige Löschung |
| **Zahlungsbelege** | 10 Jahre | § 147 AO (Steuerrecht) | Automatische Löschung |
| **Audit Logs** | 2 Jahre | Sicherheit | Automatische Löschung |
| **E-Mail Logs** | 30 Tage | Technische Notwendigkeit | Rolling deletion |
| **Analytik (anonym)** | 2 Jahre | Produktverbesserung | Automatische Löschung |
| **Gelöschte Konten** | 90 Tage (Soft Delete) | Wiederherstellung | Hard Delete nach 90d |
| **IP-Adressen (Logs)** | 7 Tage | Sicherheit | Automatische Löschung |

**Automatisierung:**
- Cronjob läuft täglich um 3:00 AM UTC
- Prüft Ablaufdaten
- Löscht abgelaufene Daten
- Loggt Löschungen (Audit Trail)

## User Rights Implementation

### Technische Umsetzung der Nutzerrechte

#### 1. Data Access (Art. 15)

**API Endpoint:**
```python
GET /users/me/data-export
```

**Response (JSON):**
```json
{
  "user": {
    "id": "uuid",
    "email": "user@example.com",
    "created_at": "2025-01-15T10:00:00Z"
  },
  "study_materials": [...],
  "exam_topics": [...],
  "review_sessions": [...],
  "subscriptions": [...]
}
```

**Download-Link:**
- Gültig für 24 Stunden
- Verschlüsselt (AES-256)

---

#### 2. Data Rectification (Art. 16)

**API Endpoint:**
```python
PATCH /users/me
{
  "full_name": "Corrected Name",
  "email": "new@email.com"  # Requires email verification
}
```

**UI:**
- Account Settings → Edit Profile

---

#### 3. Data Erasure (Art. 17)

**API Endpoint:**
```python
DELETE /users/me
{
  "password": "confirm_password",
  "reason": "optional_reason"
}
```

**Process:**
1. User requests deletion
2. Soft Delete (90 days grace period)
3. Email sent: "Your account will be deleted in 90 days. Cancel: [link]"
4. Hard Delete after 90 days:
   - Remove all personal data
   - Anonymize audit logs (`user_id = NULL`)
   - Cancel Stripe subscription
   - Delete files from storage

**Exceptions (No deletion):**
- Zahlungsbelege (10 years, legal requirement)
- Anonymized analytics data

---

#### 4. Data Portability (Art. 20)

**Format:** JSON (maschinenlesbar)

**Included Data:**
- Study materials (original + AI summaries)
- Topics and reviews
- Statistics
- Subscription history

**Not Included:**
- Passwords (security)
- Payment details (stored at Stripe)

---

#### 5. Object to Processing (Art. 21)

**Marketing Opt-Out:**
```python
PATCH /users/me
{
  "preferences": {
    "email_marketing": false
  }
}
```

**Unsubscribe Link in Email:**
```
[Abbestellen] → app.examai.com/unsubscribe?token=xyz
```
